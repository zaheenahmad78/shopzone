# 🛒 ShopZone — Full Stack E-Commerce App

A full-stack e-commerce web application built with React.js, Node.js, Express.js, and MongoDB.

## 🌐 Live Demo
👉 [shopzone-eta.vercel.app](https://shopzone-eta.vercel.app)

## 🛠️ Tech Stack

**Frontend:** React.js, TailwindCSS, React Router, Axios  
**Backend:** Node.js, Express.js, JWT Authentication, bcryptjs  
**Database:** MongoDB Atlas, Mongoose  
**Deployment:** Vercel (Frontend), Render (Backend)  

## ✨ Features

- 🔐 User Registration & Login with JWT Authentication
- 🛍️ Product Listing with Images & Prices
- 🛒 Cart — Add/Remove items, Quantity control
- 📱 Fully Responsive Design
- ☁️ Cloud Deployed & Live
